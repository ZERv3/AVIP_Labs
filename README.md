# AVIP_Labs

Небольшое учебное приложение для лабораторных работ по обработке изображений: загрузка изображений (локально и через API), преобразования цветов, бинаризация, фильтрация и выделение контуров. Включает готовые отчёты по лабораторным и демонстрацию результатов.

## Установка и запуск

1. `python -m venv .venv`
2. Активация окружения:
   - macOS/Linux: `source .venv/bin/activate`
   - Windows: `.venv\\Scripts\\activate`
3. `pip install -r requirements.txt`
4. `python main.py`
